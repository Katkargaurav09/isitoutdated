# isitoutdated.com

> A new tool for the obsolete.

**Is your stuff still worth it?** Paste any course, video, article, or book — and we'll tell you if the information is still accurate today, or if it's quietly gone stale.

🌐 **Live site:** [isitoutdated.com](https://isitoutdated.com)

---

## 🎯 What it does

Ever bought a course only to find out it teaches outdated tech? Watched a tutorial that uses tools nobody uses anymore? Read a book full of advice that no longer applies?

**isitoutdated.com** is an AI-powered tool that checks if the content INSIDE any course, video, book, or article is still relevant today. It gives you a clear verdict:

- 🟢 **Still Good** — Information is current, go for it
- 🟡 **Mostly OK** — Core ideas hold up, some parts need updating
- 🔴 **Outdated** — Skip it, here's why and what to use instead

## 💡 The Idea

Most content on the internet ages badly. A 2019 React tutorial. A 2020 SEO guide. A 2018 book on social media marketing. People still buy these, watch these, and trust these — without realizing the information is no longer accurate.

**There's no easy way to check if content is still current** before spending your time and money on it.

This project aims to fix that, using AI to read content and cross-check key claims against today's reality.
